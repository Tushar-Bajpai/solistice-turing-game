SOLSTICE://TURING
June Solstice Game Jam 2026
Project Overview
SOLSTICE://TURING is a browser-based retro terminal puzzle experience inspired by:


June Solstice

Alan Turing

Binary Logic

Cryptography

Artificial Intelligence

Light vs Darkness
The player enters a corrupted computing system and must restore balance between Light (1) and Darkness (0) by repairing four system cores.
As each core is restored, fragments of Alan Turing's legacy are recovered.
The final objective is to reconstruct Alan Turing's digital portrait and restore the system.
Core Narrative
Date: June 21
System: SOLSTICE://TURING
Event: A solar anomaly has corrupted the world's oldest AI archive.
Corruption Level: 92%
The archive contains the digital memory of Alan Turing.
Players must:


Restore Logic Core

Restore Memory Core

Restore Cipher Core

Restore AI Core
After restoration, Alan Turing's portrait can be reconstructed.
The Solstice balance is restored.
